# photo_scene

- Framework: Nerfstudio
- Method: Splatfacto
- Training iterations: 30,000
- GPU: NVIDIA GeForce RTX 4090
- Input: Custom multi-view photographs
- Camera poses: Estimated with COLMAP

## Files

- `config.yml`: Nerfstudio training configuration
- `nerfstudio_models/`: Trained model checkpoint
